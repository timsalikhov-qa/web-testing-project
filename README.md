# Web Testing - OrangeHRM Demo

Ручное тестирование веб-приложения OrangeHRM Demo.

## Тест-план

**Объект тестирования:** https://opensource-demo.orangehrmlive.com/

**Цели:**
- Проверка функциональности модуля авторизации  
- Тестирование UI/UX соответствия  
- Проверка кросс-браузерной совместимости  
- Исследовательское тестирование ключевых сценариев  

**Техники тест-дизайна:** Граничные значения, Классы эквивалентности, Таблицы решений

## Найденные дефекты

### Дефект #1: Отсутствие валидации на XSS в поле Username  
**Описание:** В поле Username возможно внедрение XSS-скрипта  
**Шаги воспроизведения:** Ввести `<script>alert('xss')</script>` в поле Username  
**Ожидаемо:** Валидация или экранирование скрипта  
**Фактически:** Скрипт выполняется (в зависимости от конфигурации сервера)  
**Серьезность:** Medium  

### Дефект #2: Несоответствие UI в мобильной версии  
**Описание:** На устройствах с шириной экрана 375px кнопка Login выходит за границы контейнера  
**Серьезность:** Low  
**Скриншот:** ![Mobile UI Issue](https://via.placeholder.com/375x667?text=Login+button+overflow+on+mobile)
